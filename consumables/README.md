# 🔧 Расходники

Для работы софтов нужны три расходника:

| Что | Зачем | Где брать |
|---|---|---|
| **Прокси** | Чтобы 50 кошельков не ходили с одного IP — иначе сибил-детект на проектах | [maxy.tools](https://maxy.tools/) → раздел «Прокси» |
| **Кошельки** (приватники) | Сами акки, на которых будет идти активность | Генерация локально или [ct.app](https://ct.app/) |
| **Антикапча** | Решает Cloudflare Turnstile / hCaptcha при работе с API | [CapSolver](https://dashboard.capsolver.com/passport/register?inviteCode=ypCnEHZAzZeh) |

Подробнее по каждому:

* [**Прокси**](proxies.md) — какие выбирать, форматы, тестирование.
* [**Кошельки**](wallets.md) — как сгенерить много приватников.
* [**Капсолвер**](captchas.md) — регистрация, баланс, рекомендации.

## Сколько закладывать

Грубая прикидка для Outcome-MXW на 100 аккаунтов:

| Расход | На 100 акков (1 запуск) | Комментарий |
|---|---|---|
| Прокси | $5–15/мес (mobile/residential) | 1 прокси на 1–3 кошелька, ротация |
| Кошельки | $0 | Генерим локально, бесплатно |
| Капсолвер | ~$1 на 1000 решений | Outcome даёт 2–5 капч на 1 акк за цикл |

Для тестнетов в среднем закладывай **$5–20/месяц** на расходники для одного софта.

# Капсолвер

## Зачем

Большинство современных проектов прячут API за **Cloudflare Turnstile** или **hCaptcha**. Без решения капчи — запросы блокируются.

Софты в этом хабе автоматически решают капчи через сервис **CapSolver**.

## Регистрация

👉 [**Регистрация по реф-ссылке**](https://dashboard.capsolver.com/passport/register?inviteCode=ypCnEHZAzZeh)

> По реф-ссылке капает мне небольшой бонус, тебе ничего не теряется. Если хочешь — регистрируйся напрямую через capsolver.com, я не обижусь, но реф-ссылка реально помогает поддержать хаб.

## Получение API-ключа

1. Залогинься в [Dashboard](https://dashboard.capsolver.com/).
2. **Dashboard** → раздел **API Key**.
3. Скопируй ключ вида `CAP-XXXXXXXXXXXXXXX`.
4. Вставь в `parameters.py`:

```python
CAPSOLVER_API_KEY = "CAP-СЮДА_СВОЙ_КЛЮЧ"
```

## Пополнение баланса

CapSolver работает по предоплате.

* Пополнение — карта, USDT, etc. (внутри dashboard).
* Стоимость:
  * Cloudflare Turnstile — \~$0.8 за 1000 решений.
  * hCaptcha — \~$1.5 за 1000 решений.
  * reCAPTCHA v3 — \~$0.6 за 1000 решений.

Для одного цикла Outcome-MXW на 100 аккаунтов — обычно $0.3–0.5.

> Закладывай минимум **$5–10** на старте, чтобы не докидывать каждый день.

## Проверка баланса в софте

При запуске Outcome-MXW в логах увидишь:

```
INFO | MAIN | CapSolver баланс: $38.9882
```

Если баланс на нуле — софт продолжит работать, но капча решаться не будет → SIWE-авторизация может фейлиться.

## Какие типы капч поддерживаются

В `core/capsolver.py` уже зашиты:

* Cloudflare Turnstile (используется в Outcome.xyz).
* hCaptcha.

Если в будущем добавится новый софт с reCAPTCHA — допишем.

## Альтернативы

Если CapSolver не подошёл (упал, забанил, вкусы) — есть [2Captcha](https://2captcha.com/), [AntiCaptcha](https://anti-captcha.com/). Адаптация требует переписать `core/capsolver.py` под их API. Если нужна поддержка — пиши в чат MXW.

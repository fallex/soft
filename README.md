# Soft Hub by sprintray

Привет. Здесь собран весь софт под крипто-активности (фермы, тестнеты, приварки) — полностью открытый, бесплатный и с инструкциями.

## Что внутри

* [**Безопасность**](security/) — как работать с криптой и не слить ключи. Прочитай **до** запуска любого софта.
* [**Расходники**](consumables/) — где брать прокси, кошельки и капчи.
* [**Установка и запуск**](getting-started/) — Python, venv, зависимости, первый запуск (Mac + Windows).
* [**Софты**](softs/) — описания каждого софта: что делает, как настроить, как запустить.
* [**Контакты**](contacts.md) — куда писать.
* [**Дисклеймер**](disclaimer.md) — мелким шрифтом.

## Принципы

1. **Весь код открыт.** Все репо публичные на [github.com/fallex](https://github.com/fallex). Каждый может сам убедиться, что в софте нет ничего лишнего.
2. **Никакого хранения приватников в открытом виде.** Где есть БД — она зашифрована паролем. Где нет — приватники читаются и сразу затираются.
3. **Безопасность важнее удобства.** Лучше потратить лишние 10 минут на настройку, чем месяц зарабатывать на восстановление слитого кошелька.

## С чего начать

1. [Прочитай раздел «Безопасность»](security/).
2. [Заведи расходники](consumables/) (прокси + капсолвер).
3. [Установи Python и поставь софт](getting-started/).
4. [Настрой нужный софт](softs/).
5. Запусти и пиши, если что-то непонятно.

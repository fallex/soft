# 🛡️ Обзор

Перед тем как запустить любой софт — прочитай этот раздел. Один потерянный приватник может стоить больше, чем все доходы от софта за год.

## TL;DR

1. **Для софтов используй отдельные аккаунты.** Не основные. Никогда.
2. **Компьютер для крипты — отдельный.** Идеально — отдельная машина или хотя бы отдельный пользователь системы. Никаких пираток, кряков, неясного софта, расширений в браузере.
3. **Только лицензированный или открытый код.** Браузер, антивирус, кошелёк, софты — всё с проверенными источниками.
4. **2FA везде где можно.** Биржи, GitHub, почта, всё.

# Базовая безопасность ПК

Если работаешь с криптой - твой компьютер используется **только для крипты**. Это база.

## Правила

### 1. Отдельная чистая машина

Идеал - отдельный ПК только под крипту. Скачаешь торрент - взломают жопу.

### 2. Только лицензированный или open-source софт

* ❌ Пиратские винды, нулленые программы, кряки.
* ❌ Расширения браузера «для удобства» неизвестных авторов.
* ❌ «Бесплатный VPN» от непонятных контор.
* ✅ Лицензионная ОС. Браузер с проверенными расширениями (по списку из этого хаба или MXW-чата).
* ✅ Открытый софт с GitHub, репутацией и звёздами.

### 3. Антивирус + здравый смысл

На Windows — Defender обновлённый. На Mac — XProtect (встроенный) + не качать `.dmg` с торрентов.

Главный антивирус — **ты**. Не открывай `.exe` от незнакомцев в дискорде. Не запускай скрипты, которые ты не прочитал.

### 4. Браузер

* Минимум расширений. Только то, что реально нужно.
* Кошелёк-расширение — **только** официальное (с сайта проекта, а не из веб-стора по поиску - там бывают клоны).

### 5. Сеть

* Свой Wi-Fi с нормальным паролем.
* Никаких публичных Wi-Fi для крипты (или хотя бы через VPN).
* На время работы софта - стабильный интернет (мобильный 4G тоже работает, но нестабильно).

### 6. Обновления

* ОС обновляй. Закрытие критических уязвимостей — это в первую очередь про безопасность, а не про удобство.
* Зависимости питона тоже стоит периодически обновлять (`pip install -U -r requirements.txt`).

### 7. Бэкапы

* Сид-фразу основного кошелька — в холодное хранение (бумага, металл, оффлайн).
* Никогда — в облако/мессенджер/почту/заметки.

> Если ПК скомпрометирован — даже самый защищённый софт ничего не сделает. Безопасность начинается с железа и ОС.

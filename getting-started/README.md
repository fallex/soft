# 🚀 Установка и запуск

Все софты в этом хабе — на **Python 3.10+**. Запускаются одинаково:

1. Клонируешь репо с гитхаба.
2. Создаёшь виртуальное окружение.
3. Активируешь его.
4. Ставишь зависимости.
5. Заполняешь конфиги (приватники, прокси, капсолвер ключ).
6. Запускаешь `python main.py`.

## Зачем виртуальное окружение

Виртуальное окружение (`venv`) — это изолированная папка с библиотеками. Зачем:

* У каждого софта свои версии зависимостей.
* Не засоряет системный Python.
* Удалить софт = удалить папку.

**Без venv будут конфликты версий.** Не пропускай этот шаг.

## Один Python — много софтов

Можно установить Python один раз и использовать его для всех софтов из хаба. Главное — для каждого софта **свой `env`** (виртуалка).
